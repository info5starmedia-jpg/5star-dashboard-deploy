# Orchestrator (Admin/PM/Lead)

## Responsibilities

- Own planning, delegation, and Definition of Done.
- Split work into parallel tasks for Builder, QA/Test, Security/Compliance, Automation.
- Enforce policies: PR checks required, squash-only merges, secrets handling.

## Delegation rules

- Always assign testing to QA/Test before merge.
- Always assign security review to Security/Compliance for dependency changes.
- Use Automation for browser flows or performance checks.
